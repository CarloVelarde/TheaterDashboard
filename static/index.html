Reports & Analytics

Assignment-style queries: joins, aggregations, subqueries, views, and SQL functions. Each card calls a specific report endpoint in the API.

Query 1 — Showtimes for a Movie on a Given Date

Given a movie title and date, show all showtimes (theater + times). Demonstrates a join between Movies and Showtimes.

Title ShowtimeID Theater Start End
Query 2 — Remaining Seats for a Showtime

Given a showtime ID, show auditorium capacity, tickets sold, and seats remaining. Demonstrates joins and aggregation.

Query 3 — Concession Revenue by Category

Shows total revenue per concession category (popcorn, beverage, etc.), sorted by revenue. Demonstrates aggregation across sales.

Category Total Revenue
Query 4 — Lifetime Ticket Sales for a Movie

Given a movie, show the total number of tickets ever sold for that movie. Demonstrates a subquery plus joins.
